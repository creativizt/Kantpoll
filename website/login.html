Transparent, secure and anonymous elections

This site (kantcoin.org) does not store any information about you. However, to interact with election campaigns, you must choose a login provider, which will store your username (phone, email, or document) and a hash code. Before signing up, create a vault and save it in a safe place. Use a strong password. You are solely responsible for the security of your data.
